<h1 align="center">Conductor</h1>

<p align="center">
  A terminal workspace for developers who use AI coding assistants.
</p>

<p align="center">
  Conductor gives Claude Code and your shell a persistent, multi-pane home so you can run several AI sessions side-by-side, keep them alive across app restarts, and see your files, git history, and browser in the same window.
</p>

---

## Features

- **Keeps Claude Code running in the background** — Sessions are backed by a persistent daemon. Close the app, reopen it, and pick up exactly where you left off with full scrollback.
- **One interface for everything** — Split panes to run Claude Code, view files, git trees, and web pages side-by-side without switching windows.
- **Tab notes** — Add notes to any tab to keep track of where you are and what each session is doing.
- **Kanban board extension** — Connect to Jira, Gitea, or Azure DevOps and autonomously work on multiple tickets in parallel.
- **Multiple Claude accounts** — Switch between API keys in one click — personal, work, client project.
- **Autopilot** — Auto-responds to CLI permission prompts and retries API errors with exponential backoff so long-running agents never stall.
- **Built in editor** — Syntax highlighting for 30+ languages.
- **File explorer** — Browse project files, preview Markdown, images, spreadsheets, Word docs, and PowerPoint slides.
- **Project files** — Save and restore your entire workspace layout in a single `.conductor` file.
- **Extension system** — Plugin architecture for adding sidebar panels, tab types, skills, and menu items.
